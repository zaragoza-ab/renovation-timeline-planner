Swedish Renovation Timeline Reference

Open reference dataset of realistic timelines for Swedish renovation projects. Covers bathroom (badrum), kitchen (kök), roof (tak), extension (tillbyggnad), and whole-house renovations. Each timeline includes phase breakdowns, typical durations in weeks, dependencies, and bygglov (building permit) requirements per Plan- och bygglagen (PBL).
